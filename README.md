# Use

2. nodejs
3. express
4. typeorm
5. typescript
6. mysql


# Start
First, check and update `ormconfig-open.js`
```
$ vi ormconfig-open.js
$ mv ./ormconfig-open.js ./ormconfig.js
$ npm i
$ npm start
```
